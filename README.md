# Trialidade Visuals

Source code and experiments behind the visuals created for **Trialidade**, an album by Marcelo USB.

This repository contains the Python scripts used inside TouchDesigner to create a custom computer vision pipeline using OpenCV for real-time visuals.

## Features

- Blob Tracking with OpenCV
- Typography following detected blobs
- TouchDesigner Script TOPs
- Real-time visual generation
- Computer Vision
- Creative Coding

## Technologies

- TouchDesigner
- Python
- OpenCV
- NumPy
